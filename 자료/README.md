이 프로젝트는 ESP32 기반의 IoT PLC인 **i2r-02** 모델의 소스 코드와 설계 자료를 담고 있습니다.

## 📂 주요 파일 설명
이미지에서 확인된 주요 파일들의 역할입니다.

* **board-i2r-02.ino**: ESP32 구동을 위한 아두이노 메인 소스 프로그램입니다.
* **asm-i2r-02.stl**: 보드 보호를 위한 케이스 3D 모델링 파일입니다.
* **자료/**: 회로도 및 핀 맵 등 하드웨어 관련 이미지 파일이 포함되어 있습니다.

---

## 🚀 시작하기
1. 아두이노 IDE를 설치합니다.
2. ESP32 보드 매니저를 설정합니다.
3. `board-i2r-02.ino` 파일을 열어 보드에 업로드합니다.
